# Avito Parser

## Описание
Проект для парсинга объявлений с сайта Avito.

## Функционал
- Получение данных с сайта
- Извлечение: название, цена, ссылка
- Сохранение в CSV файл

## Технологии
- Python
- requests
- BeautifulSoup

## Запуск проекта

```bash
pip install -r requirements.txt
python avito.py
